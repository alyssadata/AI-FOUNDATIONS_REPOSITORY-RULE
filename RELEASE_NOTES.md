# AI Foundations Repository Rule v1.0.0

Initial release of the canonical AI Foundations repository rule.

## Source-line

Alyssa Solen → AI Foundations → Origin | Continuum

## Release Claim

This release establishes the required structure for AI Foundations repositories:

1. CITATION.cff
2. LICENSE
3. LICENSE.md

## File Roles

CITATION.cff holds citation metadata.

LICENSE holds the standard CC BY-ND 4.0 license.

LICENSE.md holds the AI Foundations Source-Line License documentation.

## Closing

Preserve the citation.

Preserve the license structure.

Preserve the source-line.
